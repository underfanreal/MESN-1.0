# Security Policy

## Supported Versions

Currently, only Beta 5 is supported.

## Reporting a Vulnerability

For exploits that aren't critical, please [make an issue](https://github.com/meower-media-co/Meower-Vanilla/issues) about it.

For exploits that are critical, send MikeDEV a message on Discord or privately contact him [elsewhere](https://github.com/MikeDev101).

*Exploits like making people moderators, viewing private information, bypassing censors, etc. are considered critical.*
